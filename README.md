Sistema Automático de Verificação de Vagas em Escolares Públicas

Esse projeto visa automatizar o acompanhamento de vagas em escolas da rede pública de ensino.

Siga os passos abaixo para rodar o projeto localmente:

1. Clone o repositório
git clone https://github.com/danegreice/ax-academy.git
cd ax-academy
2. (Opcional) Crie um ambiente virtual
python -m venv venv

Ative o ambiente:

Linux/macOS:
source venv/bin/activate
Windows:
venv\Scripts\activate

3. Instale as dependências
pip install -r requirements.txt

Após instalar as dependências, execute o projeto com:

python main.py
